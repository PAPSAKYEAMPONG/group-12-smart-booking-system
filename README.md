## Project Description
A web-based Smart Booking System designed to help small businesses eliminate manual scheduling errors, prevent double-bookings, and provide a streamlined admin dashboard for appointment management.

## Tech Stack list
- **Frontend:** React.js (Vite)
- **Backend:** Firebase Cloud Functions / NoSQL
- **Database:** Firestore (Real-time DB)
- **Authentication:** Firebase Authentication
- **State Management:** React Hooks
- **Design:** Figma

## Key Features
Intelligent Booking: A 5-step guided reservation process with real-time specialist availability.

Admin Command Center: A dedicated dashboard for salon owners to track daily appointments and manage staff.

Data Integrity: Robust client-side validation using Regex to ensure clean contact data.

Responsive Branding: A fully responsive "Lively" theme optimized for both mobile and desktop users.

## Project Structure
/src/components - Reusable UI elements (Navbar, Footer, Modals).

/src/pages - Main views (Home, Booking Flow, Admin Dashboard).

/src/firebase.js - Configuration and database initialization.

/public - Static assets and branding images.

## How to Run
Clone the repo: git clone [repository-link]
